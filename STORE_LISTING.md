## Full description (4000 chars)

Apex Run is the data-driven training app for runners who treat every mile as a
science experiment. Train with the same intelligence elite athletes get from
$3,000 sensors — using only the phone in your pocket.

**Real-time GPS tracking**
- Precision distance + pace + elevation
- Custom Mapbox style with dark-mode, beautiful share-ready route renders
- Background tracking on iOS + Android with foreground service safety
- Privacy shroud blurs the start + end of your route near home

**On-device form analysis** *(rare in the category)*
- MediaPipe pose estimation analyses ground contact time + vertical oscillation
- TFLite injury-risk model flags overload before it sidelines you
- All processing on-device — frames never leave your phone

**AI coach that adapts to YOU**
- Personalized race plans (5K → marathon) generated by Gemini
- Daily insights based on your last 7 days of training
- Recovery score (0-100) from HRV + sleep + acute-chronic workload
- Audio coach calls out per-km splits + pace targets during your run

**Built for retention, not vanity**
- Streaks with freezes (Pro)
- Weekly auto-enrol challenges (25 km / week, 3 runs / week)
- 39-badge achievement catalog: lifetime mileage, PRs, hill climbs, social
- Friend feed with kudos
- Shareable 1080×1920 highlight reels for Instagram + TikTok

**Apex Pro — $9.99/mo or $79.99/yr (≈ $6.66/mo)**
- Unlimited form analysis
- AI race plans + daily personalized workouts
- Audio coach
- Streak freezes
- Unlimited history

**Apex Pro+ — $19.99/mo or $149.99/yr**
- Everything in Pro

Subscriptions auto-renew until cancelled. Cancel any time in App Store / Play

Privacy: full export + delete in-app. Health data encrypted on device, never

## Privacy questionnaire answers (App Store)

| Data | Linked to user | Used for tracking | Notes |
|---|---|---|---|
| Email address | yes | no | Auth via Supabase |
| User ID | yes | no | Supabase auth.uid |
| Health & Fitness | yes | no | HRV + sleep + heart rate via HealthKit |
| Coarse location | yes | no | Activity recording only |
| Precise location | yes | no | Activity recording only |
| Crash data | no | no | Sentry (PII stripped) |
| Performance data | no | no | Sentry traces |
| Other diagnostic data | no | no | Logger w/ redaction |

NOT collected: Contacts, Browsing History, Search History, Identifiers
(IDFA / GAID), Purchases (RevenueCat handles), Photos, Camera frames
(processed on-device only, never uploaded).

## Play Store data safety form

- **Personal info**: name, email — collected, encrypted in transit, user can delete
- **Location**: approximate + precise — collected during recording only
- **Health & Fitness**: HRV, heart rate, sleep, weight — collected, encrypted, user can delete
- **App activity**: in-app actions — collected for analytics (Sentry)
- **App info & performance**: crash logs — collected, no PII

Data deletion link: `https://apexrun.app/account/delete` (also in-app under
Profile → Delete Account).
